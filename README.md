### **Interview**

#### 1. Compile frontend
```bash
npm install
npm run build
``` 

#### 2. Run frontend
automatic compile code
```bash
npm run watch 
``` 
run development server
```bash
npm run server 
``` 
browser
```bash
http://localhost:8090
``` 

#### 2. Run frontend by webpack-dev-server
automatic compile code and run development server
```bash
npm run serve
``` 
browser
```bash
http://localhost:8090
```
